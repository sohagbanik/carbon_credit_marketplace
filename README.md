# 🌱 Carbon Credit Marketplace (Permissionless DApp)

A fully decentralized, permissionless Carbon Credit Marketplace built on the Stellar blockchain using Soroban smart contracts.

This platform allows anyone to list, buy, and trade carbon credits without any central authority — ensuring transparency, security, and trust.

🔗 Live Demo : https://carbon-credit-marketplace-sage.vercel.app/
---

## 🚀 Project Description

The Carbon Credit Marketplace is a blockchain-based decentralized application (dApp) that enables open trading of carbon credits.

Unlike traditional systems, this marketplace is **completely permissionless**, meaning:
- No admin controls
- No approval required
- Anyone can participate freely

Built using:
- 🧠 Soroban Smart Contracts (Rust)
- 🌐 Next.js Frontend
- ⚡ Stellar Blockchain

---

## ⚙️ What It Does

This dApp allows users to:

- 🌿 Create carbon credits
- 📤 List credits for sale
- 🛒 Buy carbon credits from others
- 🔄 Transfer ownership automatically via smart contracts
- 🔍 View available listings in real-time

All transactions are:
- Immutable
- Transparent
- Trustless

---

## ✨ Features

- 🔓 Fully Permissionless (No admin functions)
- 🌍 Decentralized Marketplace
- ⚡ Fast & Low-cost transactions (~5s finality, <$0.01)
- 🔐 Secure Smart Contract execution
- 📊 Real-time listing & purchase system
- 💼 Wallet integration (Connect feature)
- 🧾 Ownership tracking on-chain
